# 找猫大师 (Cat Finder) - 微信小游戏

一款基于 Canvas 2D 的逻辑解谜微信小游戏。

## 游戏规则

- 5x5 网格中放置猫咪
- 每行、每列、每个颜色区域只能有 1 只猫
- 猫的上下左右相邻位置不能有其他猫
- 玩家有 2 次错误机会
- 双击格子放置/移除猫
- 单击格子切换辅助标记（灰色圆点）

## 操作说明

- **双击空格**：放置猫咪
- **双击已有猫的格子**：移除猫咪
- **单击空格**：切换标记点
- **长按空格**：切换标记点

## 按钮功能

1. **色盲模式**：切换区域纹理背景
2. **清除标记**：清除所有灰色圆点
3. **坐标**：显示/隐藏格子内行列号
4. **第N关**：加载下一关卡
5. **提示**：高亮一个可放置猫咪的位置

## 如何运行

1. 打开微信开发者工具
2. 点击"导入项目"
3. 选择 `Cat-Finder` 文件夹
4. AppID 输入框可以留空或使用测试AppID
5. 点击"导入"即可运行游戏

## 文件说明

- `game.js` - 游戏主逻辑文件
- `game.json` - 游戏配置文件
- `cat-finder.html` - 浏览器版本（可直接用浏览器打开测试）

## 音效说明

游戏中包含5种音效：
- correct - 正确放置
- error - 错误放置
- victory - 通关
- fail - 失败
- click - 按钮点击

如需替换真实音效文件，请将音频文件放入 `assets/audio/` 目录，并修改 `game.js` 中的 `AudioManager.init()` 方法中的文件路径。

## 关卡设计

共10个关卡：
- 4x4：关卡1-2
- 5x5：关卡3-4（第4关是原始参考关卡）
- 6x6：关卡5-6
- 7x7：关卡7-8
- 8x8：关卡9-10

每关需要放置的猫数量等于网格尺寸。
